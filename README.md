# Web Tools Collection

一个包含5个实用Web工具的项目集合，采用隐私优先的设计理念。

## 项目列表

1. **privacy-toolbox** - 隐私优先的 All-in-One 工具箱（PDF、图片、文本工具）
2. **ai-tools-nav** - AI工具导航与评测网站
3. **markdown-editor** - 极简在线Markdown编辑器
4. **batch-image-processor** - 批量图片处理工具
5. **dev-toolkit** - 开发者工具集合站

## 技术栈

- React / Vue.js
- TypeScript
- Tailwind CSS
- WebAssembly（用于本地文件处理）
- Vite
