# Dev Toolkit

A comprehensive collection of free online developer tools that run entirely in your browser. No data is sent to any server, ensuring your sensitive information remains private and secure.

## 🚀 Features

- **12 Developer Tools** - All essential tools for daily development
- **100% Browser-based** - No server processing, complete privacy
- **PWA Support** - Works offline, install as app
- **Dark/Light Theme** - Choose your preferred color scheme
- **VS Code-like Editor** - Powered by Monaco Editor
- **Keyboard Shortcuts** - Efficient navigation with shortcuts
- **SEO Optimized** - Each tool has proper meta tags

## 🛠️ Tools Included

1. **JSON/XML/YAML Converter** - Convert between formats with validation
2. **Regex Tester** - Test and debug regular expressions with real-time highlighting
3. **Base64/URL Encoder-Decoder** - Encode and decode strings and files
4. **SQL Formatter** - Format SQL queries with syntax highlighting
5. **CSS/JS Minifier** - Minify and beautify code
6. **Git Command Generator** - Generate Git commands with explanations
7. **API Tester** - Lightweight Postman alternative
8. **JWT Decoder** - Decode and inspect JWT tokens
9. **UUID Generator** - Generate UUID v4 and v1 identifiers
10. **Timestamp Converter** - Convert between Unix timestamps and dates
11. **HTML Entities** - Encode and decode HTML entities
12. **Cron Expression Generator** - Build and understand cron schedules

## 🏗️ Tech Stack

- **React 18** - UI framework
- **TypeScript** - Type safety
- **Vite** - Build tool
- **Tailwind CSS** - Styling
- **Monaco Editor** - Code editing
- **React Router** - Navigation

## 📦 Installation

```bash
# Clone the repository
git clone https://github.com/yourusername/dev-toolkit.git
cd dev-toolkit

# Install dependencies
npm install

# Start development server
npm run dev

# Build for production
npm run build
```

## 🔧 Development

```bash
# Run linter
npm run lint

# Preview production build
npm run preview
```

## 📱 PWA

The app can be installed as a Progressive Web App:

1. Open the app in Chrome/Edge
2. Click the install icon in the address bar
3. Or use the "Add to Home Screen" option on mobile
