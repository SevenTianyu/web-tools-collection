# Privacy Toolbox

隐私优先的 All-in-One 工具箱。

## 特性

- 🔒 **100% 本地处理** - 所有数据处理均在浏览器本地完成，永远不会上传到服务器
- 📱 **PWA 支持** - 支持离线使用，可安装到主屏幕
- ⚡ **快速高效** - 利用浏览器原生能力，处理速度快
- 🎨 **精美界面** - 现代化的 UI 设计，支持暗色模式
- 🆓 **完全免费** - 所有功能完全免费，无需注册

## 工具列表

### PDF 工具
- PDF 合并
- PDF 拆分
- PDF 压缩
- PDF 转图片

### 图片工具
- 图片压缩
- 图片格式转换
- 图片调整大小
- 图片水印

### 文本工具
- JSON 格式化
- 代码压缩
- 正则表达式测试
- Markdown 编辑器

### AI 工具
- AI 翻译
- AI 文本总结
- AI 文本生成

## 技术栈

- React 18 + TypeScript
- Vite
- Tailwind CSS
- pdf-lib
- react-dropzone
- Monaco Editor
- Zustand

## 开发

```bash
# 安装依赖
npm install

# 启动开发服务器
npm run dev

# 构建
npm run build

# 预览
npm run preview
```
